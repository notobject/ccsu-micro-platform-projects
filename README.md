## 准备

### 系统&工具版本参考
- CentOS = 7.x
- JDK >= 1.8
- Maven >= 3.6.0
- Git >= 2.20.0
- Go >= 1.1

### 服务器资源
为能够最大程度模拟集群部署，请至少准备4台服务器/电脑：

序号|标签|用途|备注
--|--|--|--
1|ServerA|服务集群|A机房
2|ServerB|服务集群|B机房
3|ServerC|持久化集群| 这个用一台模拟
4|ServerD|服务治理平台| 这个可以选择在自己本地部署
