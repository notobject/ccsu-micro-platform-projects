## 准备

### 系统&工具版本参考
- CentOS = 7.x
- JDK >= 1.8
- Maven >= 3.6.0
- Git >= 2.20.0
- Go >= 1.1
- Nginx 
- Mysql
- Redis
- RabbitMQ

### 服务器资源分配参考
为能够最大程度模拟集群部署，请至少准备4台云服务器 or 个人电脑 or 虚拟机：

序号|标签|用途|备注
--|--|--|--
1|ServerA|服务集群|模拟A地机房
2|ServerB|服务集群|模拟B地机房
3|ServerC|持久化集群| 这个非重点，用一台模拟集群环境
4|ServerD|服务治理平台| 这个可以选择在自己本地部署
